# AIThink Assets

## Logo Image Required

Please add the AIThink logo image file here:
- File name: `AIThink_app_image.png`
- Recommended size: 256x256px or larger
- Format: PNG with transparency

This logo will be used in:
- Sidebar header
- Chat messages (AI avatar)
- Welcome screen
- Browser tab icon

If you don't have the logo yet, you can create a temporary one or use any square image as a placeholder.
